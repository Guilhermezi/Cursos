# 📚 Curso de Banco de Dados — MySQL

Repositório criado para armazenar meus estudos, exercícios e anotações do curso de **Banco de Dados** do **Gustavo Guanabara**, oferecido pela plataforma **[Curso em Vídeo](https://www.cursoemvideo.com/?utm_source=chatgpt.com)**.

## 🎯 Objetivo

Este repositório foi desenvolvido com o objetivo de documentar minha evolução no aprendizado de **Banco de Dados**, praticando desde conceitos básicos até comandos SQL mais avançados.

Durante o curso, estou aprendendo a:

* Criar bancos de dados
* Criar tabelas
* Inserir registros
* Consultar dados
* Atualizar informações
* Excluir registros
* Trabalhar com relacionamentos entre tabelas
* Entender modelagem de dados
