# Primeiro App em Kotlin 📱

Este repositório contém apenas o **arquivo Kotlin principal** do aplicativo.

O restante dos arquivos do projeto Android não foi incluído porque **não são necessários para demonstrar o código**.
A intenção deste repositório é mostrar que o aplicativo foi **desenvolvido utilizando Kotlin e Jetpack Compose**.

## Conteúdo do repositório

* `MainActivity.kt` — Arquivo principal contendo a lógica e a interface do aplicativo.

## APK para teste

Também incluí o **arquivo APK** do aplicativo para que seja possível **testar diretamente no celular Android**, sem necessidade de abrir o projeto no Android Studio.

## Como testar o aplicativo

1. Baixe o arquivo `.apk` presente no repositório.
2. Transfira para seu celular Android.
3. Abra o arquivo APK no dispositivo.
4. Permita a instalação de **fontes desconhecidas** caso o sistema solicite.
5. Instale o aplicativo.

Após a instalação, o app poderá ser executado normalmente no dispositivo.

## Tecnologias utilizadas

* **Kotlin**
* **Android Studio**
* **Jetpack Compose**
